 ### Step 1 — 引入二次项（关键洞察）

 - x=0 和 x=-5 处：残差分别为 -1.8 和 +1.3                                                            
 - x=10 和 x=15 处：残差为 +1.0 和 -1.1                                                               
                                                                                                      
 这说明曲线在 x>0 区域比线性上升更快，需要正的二次项使右侧向上弯曲。                                  
                                                                                                      
 猜测 a 为正的小数，尝试 a = 0.03：                                                                   
                                                                                                      
 用公式 y = 0.03x² + bx + c，b 和 c 相应微调（b 略减以补偿二次项在 x<0 区的副作用）：                 
                                                                                                      
 取 a = 0.03, b = 1.75, c = 2.0：                                                                     
                                                                                                      
 ┌─────┬─────────────────────┬───────┬──────┐                                                         
 │ x   │ 预测y               │ 实际y │ 残差 │                                                         
 ├─────┼─────────────────────┼───────┼──────┤                                                         
 │ -5  │ -7.3+2.0 = -5.3     │ -6.0  │ +0.7 │                                                         
 ├─────┼─────────────────────┼───────┼──────┤                                                         
 │ 0   │ +2.0                │ ~0    │ -2.0 │
